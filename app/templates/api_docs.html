{# ═══════════════════════════════ TICKETS STATUS ═══════════════════════════════ #}


  

    POST
    /api/tickets/{id}/status
    Atualiza o status de um chamado
    120 req/h
  


  

    

      
Body application/json

      

        
CampoTipoObrigatórioDescrição

        
          
statusstring✅Novo status do chamado

        
      

      
Valores de status

      

        {% for s in ["ABERTO","EM_ANDAMENTO","AGUARDANDO_FORNECEDOR","AGUARDANDO_APROVACAO","ENVIADO","CONCLUIDO","CANCELADO"] %}
          {{ s }}
        {% endfor %}
      

      
Códigos de resposta

      

        
          
200Status atualizado

          
400Status inválido

          
404Chamado não encontrado

          
429Rate limit atingido

        
      

    

    

      
Exemplo de requisição

      
POST /api/tickets/1/status
Content-Type: application/json

{
  "status": "EM_ANDAMENTO"
}

      
Resposta 200 OK

      
{
  "ok": true,
  "id": 1,
  "status": "EM_ANDAMENTO"
}

      
Resposta 400 Bad Request

      
{
  "error": "Status inválido: 'INVALIDO'."
}
